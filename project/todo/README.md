## 📝 앱 소개

‘**네모진 투두**’는 일정을 효과적으로 관리할 수 있도록 도와주는 **기본형 투두앱**입니다.

- 📅 날짜 중심으로 일정을 생성
- 🔁 반복되는 일정도 손쉽게 등록/관리
- ✨ 깔끔한 UI와 직관적인 UX로 누구나 빠르게 사용 가능
- [▶ Google Play에서 다운로드](https://play.google.com/store/apps/details?id=com.naimjae.todo.todo)

## 📝 기능 소개
### 📌 일정 등록
<img width="100%" alt="일정 등록" src="https://github.com/user-attachments/assets/e3567a47-a44e-4e41-abf6-0742bec4fb16" />

- 날짜와 시간, 일정 제목을 입력해 일정을 생성할 수 있습니다.
- 필요한 경우 세부 항목을 추가하여 더 구체적인 일정으로 만들 수 있습니다.
- 색상을 지정해 목록 화면에서 일정을 시각적으로 구분할 수 있습니다.
- 일정의 반복 설정은 매일, 매주, 매월, 특정 간격 등으로 자유롭게 구성 가능합니다.
- 반복 방식은 UI 하단 설명을 통해 직관적으로 확인할 수 있습니다.
<br>

### 📌 일정 관리
<img width="100%" alt="일정 관리" src="https://github.com/user-attachments/assets/c1b8b1d0-2a36-4592-9953-d0bdf88ca9f9" />

- 일정 목록 상단의 날짜 바를 통해 일자별 일정을 한눈에 확인할 수 있습니다.
- 좌우 화살표 버튼 또는 년/월 선택 기능으로 원하는 날짜로 빠르게 이동할 수 있습니다.
- 각 일정은 카드 형태로 정리되며, 완료 여부에 따라 진행률(%)이 표시됩니다.
- 세부 항목이 있는 일정은 모든 항목을 완료하면 자동으로 일정이 완료 처리됩니다.
<br>

### 📌 반복 일정 관리
<img width="100%" alt="반복 일정 관리" src="https://github.com/user-attachments/assets/b1c704eb-fa3c-44f3-870c-ca10d8d3a7b0" />

- 일정 목록 우측 상단의 반복 일정 버튼을 통해 등록된 모든 반복 일정을 관리할 수 있습니다.
- 반복 일정의 제목, 세부 항목, 반복 주기 등의 정보를 확인하거나 수정 및 삭제가 가능합니다.
- UI가 직관적으로 구성되어 있어, 반복 일정 관리도 쉽게 할 수 있습니다.
<br/>
